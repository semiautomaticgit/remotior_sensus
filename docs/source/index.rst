Remotior Sensus (which is Latin for "a more remote sense")
is a Python package that allows for the processing
of remote sensing images and GIS data.

**Source code:** https://github.com/semiautomaticgit/remotior_sensus

**Bug reports:** https://github.com/semiautomaticgit/remotior_sensus/issues
